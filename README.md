# Onshape-logo-experiment-old-twitter-logo-

Here i made the old Twitter logo using `Onshape`, defining dimentions, and using the `extrude` tab in oreder to make into a 3D shape, with the goal of printing it with a 3D printer as a key-chain.

The file of the finel product is uploaded in this repository as a STL file.
